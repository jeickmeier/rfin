## Key Implementation: BaseCorrelationCalibrator

### WASM Implementation

**Location:** `finstack-wasm/src/valuations/calibration/methods.rs:407-538`

**Features:**
- Constructor with 4 parameters (index_id, series, maturity_years, base_date)
- Configuration methods: `withConfig`, `withDiscountCurveId`, `withDetachmentPoints`
- Calibration method: `calibrate(quotes, market)` returns `[curve, report]`
- Comprehensive JSDoc with examples
- Proper error handling with context

**Example Usage:**
```typescript
import { 
  BaseCorrelationCalibrator,
  CreditQuote,
  FsDate 
} from 'finstack-wasm';

const calibrator = new BaseCorrelationCalibrator(
  "CDX.NA.IG.42",
  42,
  5.0,
  new FsDate(2025, 1, 1)
)
  .withConfig(config)
  .withDetachmentPoints(new Float64Array([3.0, 7.0, 10.0, 15.0, 30.0]));

const quotes = [
  CreditQuote.cdsTranche("CDX.NA.IG.42", 0.0, 3.0, maturityDate, 15.0, 500.0),
  CreditQuote.cdsTranche("CDX.NA.IG.42", 3.0, 7.0, maturityDate, 5.0, 150.0),
  // ... more tranches
];

const [curve, report] = calibrator.calibrate(quotes, market);
console.log(`Calibration ${report.success ? 'succeeded' : 'failed'}`);
console.log(`Iterations: ${report.iterations}, Residual: ${report.maxResidual}`);
```

### Python Equivalent

**Already existed:** `finstack-py/src/valuations/calibration/methods.rs:590-796`

```python
from finstack.valuations.calibration import BaseCorrelationCalibrator, CreditQuote
from datetime import date

calibrator = (BaseCorrelationCalibrator("CDX.NA.IG.42", 42, 5.0, date(2025, 1, 1))
    .with_config(config)
    .with_detachment_points([3.0, 7.0, 10.0, 15.0, 30.0]))

quotes = [
    CreditQuote.cds_tranche("CDX.NA.IG.42", 0.0, 3.0, maturity_date, 15.0, 500.0),
    CreditQuote.cds_tranche("CDX.NA.IG.42", 3.0, 7.0, maturity_date, 5.0, 150.0),
    # ... more tranches
]

curve, report = calibrator.calibrate(quotes, market)
print(f"Calibration {'succeeded' if report.success else 'failed'}")
print(f"Iterations: {report.iterations}, Residual: {report.max_residual}")
```

**Perfect Parity:** Same functionality, language-idiomatic naming.
