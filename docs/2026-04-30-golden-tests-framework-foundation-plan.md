# Golden Tests Framework — Foundation (Phases 0–2) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the golden-test framework foundation: existing-test cleanup (Phase 0), shared fixture schema and runners in both Rust and Python (Phase 1), and the first end-to-end golden — `pricing/irs/usd_sofr_5y_par.json` — passing in both layers (Phase 2).

**Architecture:** JSON fixtures conforming to schema `finstack.golden/1` live under each crate's `tests/golden/data/`. A Rust runner (`run_golden!` macro + per-domain dispatch) and a Python runner (`run_golden()` + parametrized pytest) consume the same JSON files via path-based discovery. Reference data is captured ahead of test time by adapter scripts under `scripts/goldens/`. CI runs both layers offline against committed JSON.

**Tech Stack:** Rust (cargo + serde_json), Python 3.12 (uv + pytest + QuantLib-Python ≥ 1.34), `mise` task runner, finstack workspace canonical APIs.

**Spec reference:** [2026-04-30-golden-tests-framework-design.md](2026-04-30-golden-tests-framework-design.md). Section numbers below reference that doc.

**Out of scope for this plan:**
- Phases 3–7 (remaining 88 fixtures across rates/FI/equity/FX/credit/inflation/calibration/integration/analytics/attribution/manual-source) — Plan 2.
- Phases 8–9 (drift/stale tooling, CI integration polish, reference-doc updates) — Plan 3.
- Bloomberg-screen / Intex / formula source modes — Plan 2 introduces them.

```rust
//! Serde structs for the `finstack.golden/1` fixture schema.
//!
//! See docs/2026-04-30-golden-tests-framework-design.md §5.

use serde::{Deserialize, Serialize};
use std::collections::BTreeMap;

/// Top-level fixture envelope. One per JSON file under `tests/golden/data/`.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct GoldenFixture {
    /// Schema version. Must equal `"finstack.golden/1"`.
    pub schema_version: String,
    /// Snake-case unique name within domain.
    pub name: String,
    /// Dotted domain path: `rates.irs`, `analytics.performance`, etc.
    pub domain: String,
    /// One-sentence description.
    pub description: String,
    /// Source, capture, and review metadata.
    pub provenance: Provenance,
    /// Domain-specific input object — opaque at this layer; the runner deserializes further.
    pub inputs: serde_json::Value,
    /// Map of metric name → reference value.
    pub expected_outputs: BTreeMap<String, f64>,
    /// Map of metric name → tolerance entry. Must cover every key in `expected_outputs`.
    pub tolerances: BTreeMap<String, ToleranceEntry>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Provenance {
    /// YYYY-MM-DD market date the reference values represent.
    pub as_of: String,
    /// Source mode: quantlib | bloomberg-api | bloomberg-screen | intex | formula | textbook.
    pub source: String,
    /// Free-form: QL version + engine, Bloomberg screen + ticker, textbook page.
    pub source_detail: String,
    /// Username at capture time.
    pub captured_by: String,
    /// YYYY-MM-DD when fixture was first written.
    pub captured_on: String,
    /// Username at last review.
    pub last_reviewed_by: String,
    /// YYYY-MM-DD when fixture was last reviewed.
    pub last_reviewed_on: String,
    /// Default 6. Drives the stale-report threshold.
    pub review_interval_months: u32,
    /// Exact command to regenerate. Empty allowed for `formula` / `textbook` sources.
    pub regen_command: String,
    /// Image evidence. Empty allowed for `quantlib` / `bloomberg-api` / `formula` / `textbook`. Required for `bloomberg-screen` and `intex`.
    #[serde(default)]
    pub screenshots: Vec<Screenshot>,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct Screenshot {
    /// Path relative to the fixture JSON.
    pub path: String,
    /// Bloomberg / Intex screen name (SWPM, OVME, CDU, etc.).
    pub screen: String,
    /// YYYY-MM-DD.
    pub captured_on: String,
    /// Free-form description.
    pub description: String,
}

#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ToleranceEntry {
    /// Absolute tolerance: |actual - expected| ≤ abs.
    #[serde(skip_serializing_if = "Option::is_none")]
    pub abs: Option<f64>,
    /// Relative tolerance: |actual - expected| / max(|expected|, 1e-12) ≤ rel.
    #[serde(skip_serializing_if = "Option::is_none")]
    pub rel: Option<f64>,
    /// Optional documentation when overriding default tolerance.
    #[serde(skip_serializing_if = "Option::is_none")]
    pub tolerance_reason: Option<String>,
}

pub const SCHEMA_VERSION: &str = "finstack.golden/1";
```

```rust
//! Tolerance comparator — abs OR rel passes (not both required).
//!
//! See spec §5.3.

use crate::golden::schema::ToleranceEntry;

const REL_DENOM_MIN: f64 = 1e-12;

/// Result of comparing one metric against its reference value.
#[derive(Debug, Clone)]
pub struct ComparisonResult {
    pub metric: String,
    pub actual: f64,
    pub expected: f64,
    pub abs_diff: f64,
    pub rel_diff: f64,
    pub passed: bool,
    pub used_tolerance: ToleranceEntry,
}

impl ComparisonResult {
    /// Format a failure message identifying the metric, values, diffs, and tolerance used.
    pub fn failure_message(&self, fixture_path: &str) -> String {
        format!(
            "Golden mismatch in {}\n  metric: {}\n  actual: {:.12}\n  expected: {:.12}\n  abs_diff: {:.6e}\n  rel_diff: {:.6e}\n  tolerance: abs={:?}, rel={:?}",
            fixture_path,
            self.metric,
            self.actual,
            self.expected,
            self.abs_diff,
            self.rel_diff,
            self.used_tolerance.abs,
            self.used_tolerance.rel,
        )
    }
}

/// Compare a single metric. Passes if abs OR rel tolerance is satisfied.
/// Panics if neither `abs` nor `rel` is set on the tolerance entry — that's a malformed fixture.
pub fn compare(
    metric: &str,
    actual: f64,
    expected: f64,
    tol: &ToleranceEntry,
) -> ComparisonResult {
    let abs_diff = (actual - expected).abs();
    let rel_diff = abs_diff / expected.abs().max(REL_DENOM_MIN);

    let abs_pass = tol.abs.map_or(false, |a| abs_diff <= a);
    let rel_pass = tol.rel.map_or(false, |r| rel_diff <= r);
    let any_tolerance_set = tol.abs.is_some() || tol.rel.is_some();
    assert!(
        any_tolerance_set,
        "ToleranceEntry for metric '{}' has neither abs nor rel — malformed fixture",
        metric
    );

    ComparisonResult {
        metric: metric.to_string(),
        actual,
        expected,
        abs_diff,
        rel_diff,
        passed: abs_pass || rel_pass,
        used_tolerance: tol.clone(),
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    fn abs_only(abs: f64) -> ToleranceEntry {
        ToleranceEntry { abs: Some(abs), rel: None, tolerance_reason: None }
    }
    fn rel_only(rel: f64) -> ToleranceEntry {
        ToleranceEntry { abs: None, rel: Some(rel), tolerance_reason: None }
    }
    fn both(abs: f64, rel: f64) -> ToleranceEntry {
        ToleranceEntry { abs: Some(abs), rel: Some(rel), tolerance_reason: None }
    }

    #[test]
    fn abs_only_pass() {
        let r = compare("x", 1.005, 1.0, &abs_only(0.01));
        assert!(r.passed);
    }

    #[test]
    fn abs_only_fail() {
        let r = compare("x", 1.5, 1.0, &abs_only(0.01));
        assert!(!r.passed);
    }

    #[test]
    fn rel_only_pass() {
        let r = compare("x", 1.0001, 1.0, &rel_only(1e-3));
        assert!(r.passed);
    }

    #[test]
    fn rel_handles_zero_expected() {
        // Expected = 0 makes rel ill-defined; we use REL_DENOM_MIN to stabilize.
        // Actual = 1e-15 with rel 1e-3 should pass because |1e-15| / max(0, 1e-12) = 1e-3 ≤ rel 1e-3.
        let r = compare("x", 1e-15, 0.0, &rel_only(1e-3));
        assert!(r.passed, "expected pass, got {}", r.failure_message("test"));
    }

    #[test]
    fn either_passes() {
        // abs fails (diff = 0.5, abs tol = 0.01) but rel passes (rel_diff = 5e-7).
        let r = compare("x", 1_000_000.5, 1_000_000.0, &both(0.01, 1e-6));
        assert!(r.passed);
    }

    #[test]
    fn neither_passes() {
        let r = compare("x", 100.0, 1.0, &both(0.01, 1e-6));
        assert!(!r.passed);
    }

    #[test]
    #[should_panic(expected = "neither abs nor rel")]
    fn empty_tolerance_panics() {
        let empty = ToleranceEntry { abs: None, rel: None, tolerance_reason: None };
        compare("x", 1.0, 1.0, &empty);
    }
}
```

```rust
//! Walk-test — validates every fixture under `tests/golden/data/` is well-formed.
//!
//! Runs as a regular `#[test]` so a malformed fixture committed by accident fails CI
//! before any individual fixture-driven test runs.
//!
//! See spec §7.1 ("walk-test").

use crate::golden::schema::{GoldenFixture, SCHEMA_VERSION};
use std::fs;
use std::path::{Path, PathBuf};

const DATA_ROOT: &str = "tests/golden/data";

/// Recursively collect all *.json files under DATA_ROOT.
fn collect_fixture_paths() -> Vec<PathBuf> {
    let manifest_dir = env!("CARGO_MANIFEST_DIR");
    let root = Path::new(manifest_dir).join(DATA_ROOT);
    if !root.exists() {
        return Vec::new();
    }
    let mut out = Vec::new();
    walk_dir(&root, &mut out);
    out.sort(); // deterministic order
    out
}

fn walk_dir(dir: &Path, out: &mut Vec<PathBuf>) {
    let entries = match fs::read_dir(dir) {
        Ok(e) => e,
        Err(_) => return,
    };
    for entry in entries.flatten() {
        let path = entry.path();
        if path.is_dir() {
            // Skip screenshots/ subdirectories — they hold images, not fixtures.
            if path.file_name().and_then(|s| s.to_str()) == Some("screenshots") {
                continue;
            }
            walk_dir(&path, out);
        } else if path.extension().and_then(|s| s.to_str()) == Some("json") {
            out.push(path);
        }
    }
}

fn validate_fixture(path: &Path) -> Result<(), String> {
    let raw = fs::read_to_string(path)
        .map_err(|e| format!("read failed: {}", e))?;
    let fixture: GoldenFixture = serde_json::from_str(&raw)
        .map_err(|e| format!("parse failed: {}", e))?;

    if fixture.schema_version != SCHEMA_VERSION {
        return Err(format!(
            "schema_version is '{}', expected '{}'",
            fixture.schema_version, SCHEMA_VERSION
        ));
    }

    // Provenance presence is enforced by serde Deserialize (all fields required
    // except `screenshots` which has a serde default). Spot-check non-empty:
    if fixture.provenance.captured_by.is_empty() {
        return Err("provenance.captured_by is empty".into());
    }
    if fixture.provenance.last_reviewed_by.is_empty() {
        return Err("provenance.last_reviewed_by is empty".into());
    }

    // Tolerances must cover every key in expected_outputs.
    for metric in fixture.expected_outputs.keys() {
        if !fixture.tolerances.contains_key(metric) {
            return Err(format!(
                "expected_outputs has '{}' but tolerances does not",
                metric
            ));
        }
    }
    // And tolerances should not have entries for metrics not in expected_outputs.
    for metric in fixture.tolerances.keys() {
        if !fixture.expected_outputs.contains_key(metric) {
            return Err(format!(
                "tolerances has '{}' but expected_outputs does not",
                metric
            ));
        }
    }

    // bloomberg-screen and intex sources MUST have at least one screenshot.
    let needs_screenshot = matches!(
        fixture.provenance.source.as_str(),
        "bloomberg-screen" | "intex"
    );
    if needs_screenshot && fixture.provenance.screenshots.is_empty() {
        return Err(format!(
            "source '{}' requires at least one screenshot",
            fixture.provenance.source
        ));
    }

    // Every screenshot path must exist on disk relative to the fixture JSON.
    let parent = path.parent().ok_or("fixture has no parent dir")?;
    for shot in &fixture.provenance.screenshots {
        let shot_path = parent.join(&shot.path);
        if !shot_path.exists() {
            return Err(format!(
                "screenshot '{}' does not exist (resolved to {:?})",
                shot.path, shot_path
            ));
        }
    }

    Ok(())
}

#[test]
fn all_fixtures_well_formed() {
    let fixtures = collect_fixture_paths();
    let mut failures: Vec<String> = Vec::new();
    for path in &fixtures {
        if let Err(msg) = validate_fixture(path) {
            failures.push(format!("{}: {}", path.display(), msg));
        }
    }
    assert!(
        failures.is_empty(),
        "{} fixture(s) failed validation:\n{}",
        failures.len(),
        failures.join("\n")
    );
}
```

```rust
//! Runner trait + dispatch by domain. See spec §7.3.
//!
//! Each domain (e.g. `rates.irs`, `analytics.performance`) implements `DomainRunner`
//! and registers itself in `dispatch()`. The macro `run_golden!` reads the fixture's
//! `domain` field and looks up the runner.

use crate::golden::schema::GoldenFixture;
use crate::golden::tolerance::{compare, ComparisonResult};

/// One runner per domain. Builds inputs, calls canonical API, extracts metrics.
pub trait DomainRunner {
    /// Run the canonical computation for this fixture, returning the actual values
    /// for every metric in `fixture.expected_outputs`.
    fn run(&self, fixture: &GoldenFixture) -> Result<std::collections::BTreeMap<String, f64>, String>;
}

/// Dispatch a fixture to its domain runner by `domain` field.
pub fn dispatch(fixture: &GoldenFixture) -> Result<Box<dyn DomainRunner>, String> {
    match fixture.domain.as_str() {
        "rates.irs" => Ok(Box::new(crate::golden::runners::pricing_irs::IrsRunner)),
        // Additional domain matches are added as runners land in later tasks/plans.
        other => Err(format!("no runner registered for domain '{}'", other)),
    }
}

/// Run a fixture end-to-end: dispatch → compute actuals → compare against expected.
/// Returns the per-metric ComparisonResults; caller decides how to report.
pub fn run_fixture(fixture: &GoldenFixture) -> Result<Vec<ComparisonResult>, String> {
    let runner = dispatch(fixture)?;
    let actuals = runner.run(fixture)?;
    let mut results = Vec::new();
    for (metric, expected) in &fixture.expected_outputs {
        let actual = actuals
            .get(metric)
            .copied()
            .ok_or_else(|| format!("runner did not produce metric '{}'", metric))?;
        let tol = fixture
            .tolerances
            .get(metric)
            .ok_or_else(|| format!("no tolerance for metric '{}'", metric))?;
        results.push(compare(metric, actual, *expected, tol));
    }
    Ok(results)
}

/// Macro for one-line test bodies. See spec §7.1.
#[macro_export]
macro_rules! run_golden {
    ($relative_path:expr) => {{
        let manifest_dir = env!("CARGO_MANIFEST_DIR");
        let path = std::path::Path::new(manifest_dir)
            .join("tests/golden/data")
            .join($relative_path);
        let raw = std::fs::read_to_string(&path)
            .unwrap_or_else(|e| panic!("read fixture {:?}: {}", path, e));
        let fixture: $crate::golden::schema::GoldenFixture = serde_json::from_str(&raw)
            .unwrap_or_else(|e| panic!("parse fixture {:?}: {}", path, e));
        let results = $crate::golden::runner::run_fixture(&fixture)
            .unwrap_or_else(|e| panic!("run fixture {:?}: {}", path, e));
        let mut failures: Vec<String> = Vec::new();
        for r in &results {
            if !r.passed {
                failures.push(r.failure_message(&path.display().to_string()));
            }
        }
        if !failures.is_empty() {
            panic!("{} metric(s) failed:\n{}", failures.len(), failures.join("\n\n"));
        }
    }};
}
```

```python
"""Tolerance comparator — abs OR rel passes (not both required). Mirrors Rust tolerance.rs."""
from __future__ import annotations

from dataclasses import dataclass

from .schema import ToleranceEntry

REL_DENOM_MIN = 1e-12


@dataclass
class ComparisonResult:
    metric: str
    actual: float
    expected: float
    abs_diff: float
    rel_diff: float
    passed: bool
    used_tolerance: ToleranceEntry

    def failure_message(self, fixture_path: str) -> str:
        return (
            f"Golden mismatch in {fixture_path}\n"
            f"  metric: {self.metric}\n"
            f"  actual: {self.actual:.12g}\n"
            f"  expected: {self.expected:.12g}\n"
            f"  abs_diff: {self.abs_diff:.6e}\n"
            f"  rel_diff: {self.rel_diff:.6e}\n"
            f"  tolerance: abs={self.used_tolerance.abs!r}, rel={self.used_tolerance.rel!r}"
        )


def compare(metric: str, actual: float, expected: float, tol: ToleranceEntry) -> ComparisonResult:
    abs_diff = abs(actual - expected)
    rel_diff = abs_diff / max(abs(expected), REL_DENOM_MIN)

    abs_pass = tol.abs is not None and abs_diff <= tol.abs
    rel_pass = tol.rel is not None and rel_diff <= tol.rel
    if tol.abs is None and tol.rel is None:
        raise AssertionError(
            f"ToleranceEntry for metric '{metric}' has neither abs nor rel — malformed fixture"
        )

    return ComparisonResult(
        metric=metric,
        actual=actual,
        expected=expected,
        abs_diff=abs_diff,
        rel_diff=rel_diff,
        passed=abs_pass or rel_pass,
        used_tolerance=tol,
    )
```

```python
"""pytest helpers for golden tests — runs the same JSON fixtures the Rust runners read.

Path resolution: fixtures live under each Rust crate's `tests/golden/data/`. We
construct paths from a workspace-relative root.
"""
from __future__ import annotations

import os
from pathlib import Path
from typing import Iterable

from .schema import GoldenFixture
from .tolerance import compare

# Workspace root resolved from this file's location:
# finstack-py/tests/golden/conftest.py -> .../finstack-py/.. = workspace root
WORKSPACE_ROOT = Path(__file__).resolve().parents[3]

# Map a subdomain prefix to its owning crate's data root.
# Order matters: the first matching prefix wins.
DATA_ROOTS = {
    "pricing": WORKSPACE_ROOT / "finstack/valuations/tests/golden/data",
    "calibration": WORKSPACE_ROOT / "finstack/valuations/tests/golden/data",
    "integration": WORKSPACE_ROOT / "finstack/valuations/tests/golden/data",
    "attribution": WORKSPACE_ROOT / "finstack/valuations/tests/golden/data",
    "analytics": WORKSPACE_ROOT / "finstack/analytics/tests/golden/data",
}


def _data_root_for(relative_path: str) -> Path:
    top = relative_path.split("/", 1)[0]
    if top not in DATA_ROOTS:
        raise ValueError(
            f"path '{relative_path}' does not start with a known top-level domain "
            f"(known: {list(DATA_ROOTS)})"
        )
    return DATA_ROOTS[top]


def fixture_path(relative_path: str) -> Path:
    """Resolve a relative path like 'pricing/irs/foo.json' to an absolute Path."""
    return _data_root_for(relative_path) / relative_path


def discover_fixtures(relative_dir: str) -> list[str]:
    """Return all JSON fixture paths under a relative directory, as strings.
    Used with pytest.mark.parametrize so a single test file becomes one test per fixture.
    """
    root = _data_root_for(relative_dir) / relative_dir
    if not root.exists():
        return []
    return sorted(
        str(p.relative_to(_data_root_for(relative_dir)))
        for p in root.rglob("*.json")
    )


# Domain → Python runner module path. Add a new domain by adding an entry
# (the module must export a `run(fixture) -> dict[str, float]` function).
_DOMAIN_RUNNERS = {
    "rates.irs": "finstack_py_tests.golden.runners.pricing_irs",
}


def _load_runner(domain: str):
    if domain not in _DOMAIN_RUNNERS:
        raise ValueError(f"no Python runner registered for domain '{domain}'")
    # Import lazily so a missing optional dependency in one runner doesn't
    # break unrelated golden tests.
    import importlib
    # The module path uses a synthetic root; resolve it relative to the conftest dir.
    mod_short = _DOMAIN_RUNNERS[domain].rsplit(".", 1)[1]
    return importlib.import_module(f".runners.{mod_short}", package=__package__)


def run_golden(relative_path: str) -> None:
    """Load a fixture, dispatch to its runner, compare actuals vs expected, assert all pass."""
    path = fixture_path(relative_path)
    fixture = GoldenFixture.from_path(path)
    runner = _load_runner(fixture.domain)
    actuals = runner.run(fixture)

    failures = []
    for metric, expected in fixture.expected_outputs.items():
        if metric not in actuals:
            failures.append(f"{path}: runner did not produce metric '{metric}'")
            continue
        tol = fixture.tolerances[metric]
        result = compare(metric, actuals[metric], expected, tol)
        if not result.passed:
            failures.append(result.failure_message(str(path)))
    if failures:
        raise AssertionError(
            f"{len(failures)} metric(s) failed:\n" + "\n\n".join(failures)
        )
```

```python
from pathlib import Path

from .schema import GoldenFixture, SCHEMA_VERSION


def _all_fixtures() -> list[Path]:
    out: list[Path] = []
    seen: set[Path] = set()
    for root in DATA_ROOTS.values():
        if root in seen or not root.exists():
            continue
        seen.add(root)
        out.extend(p for p in root.rglob("*.json") if "screenshots" not in p.parts)
```

```python
"""Base class for golden adapters. See spec §8.1."""
from __future__ import annotations

import json
from abc import ABC, abstractmethod
from dataclasses import asdict, dataclass
from datetime import date
from pathlib import Path
from typing import Any


SCHEMA_VERSION = "finstack.golden/1"
DEFAULT_REVIEW_INTERVAL_MONTHS = 6


@dataclass
class AdapterArgs:
    """Common arguments every adapter needs."""
    as_of: str           # YYYY-MM-DD
    reviewer: str        # username
    out: Path            # absolute path to output JSON
    source: str          # quantlib | bloomberg-api | bloomberg-screen | intex | formula | textbook
    source_detail: str   # free-form: QL version, BBG screen, textbook page


class BaseAdapter(ABC):
    """One per --kind. Subclass and implement build()."""

    @property
    @abstractmethod
    def kind(self) -> str:
        """The --kind value this adapter handles, e.g. 'irs-par'."""

    @property
    @abstractmethod
    def domain(self) -> str:
        """The fixture's domain field, e.g. 'rates.irs'."""

    @abstractmethod
    def build(self, args: AdapterArgs, kind_args: dict[str, Any]) -> dict[str, Any]:
        """Return the fixture body (everything except the envelope/provenance).
        Must include keys: name, description, inputs, expected_outputs, tolerances.
        """

    def regen_command(self, args: AdapterArgs, kind_args: dict[str, Any]) -> str:
        """Reconstruct the CLI command that produced this fixture (must be runnable)."""
        kw = " ".join(f"--kw {k}={v}" for k, v in sorted(kind_args.items()))
        return (
            f"uv run python -m scripts.goldens.regen --kind {self.kind} "
            f"--as-of {args.as_of} --reviewer {args.reviewer} "
            f"--source {args.source} {kw} --out {args.out}"
        )

    def emit(self, args: AdapterArgs, kind_args: dict[str, Any]) -> None:
        body = self.build(args, kind_args)
        envelope = {
            "schema_version": SCHEMA_VERSION,
            "name": body["name"],
            "domain": self.domain,
            "description": body["description"],
            "provenance": {
                "as_of": args.as_of,
                "source": args.source,
                "source_detail": args.source_detail,
                "captured_by": args.reviewer,
                "captured_on": date.today().isoformat(),
                "last_reviewed_by": args.reviewer,
                "last_reviewed_on": date.today().isoformat(),
                "review_interval_months": DEFAULT_REVIEW_INTERVAL_MONTHS,
                "regen_command": self.regen_command(args, kind_args),
                "screenshots": [],
            },
            "inputs": body["inputs"],
            "expected_outputs": body["expected_outputs"],
            "tolerances": body["tolerances"],
        }
        args.out.parent.mkdir(parents=True, exist_ok=True)
        args.out.write_text(json.dumps(envelope, indent=2, sort_keys=False) + "\n")
        print(f"wrote {args.out}")
```

```python
"""regen.py — generate or refresh a golden fixture. See spec §8.1."""
from __future__ import annotations

import argparse
import importlib
from pathlib import Path

from .adapters.base import AdapterArgs, BaseAdapter


# --kind → adapter module under adapters/
KIND_REGISTRY: dict[str, str] = {
    "irs-par": "scripts.goldens.adapters.irs_par",
    # New kinds added in later plans.
}


def _load_adapter(kind: str) -> BaseAdapter:
    if kind not in KIND_REGISTRY:
        raise SystemExit(f"unknown --kind '{kind}'. Known: {sorted(KIND_REGISTRY)}")
    mod = importlib.import_module(KIND_REGISTRY[kind])
    if not hasattr(mod, "ADAPTER"):
        raise SystemExit(f"adapter module {mod.__name__} must export ADAPTER instance")
    return mod.ADAPTER


def main() -> None:
    p = argparse.ArgumentParser(description="Generate or refresh a golden test fixture.")
    p.add_argument("--kind", required=True, help="Adapter kind (e.g. irs-par)")
    p.add_argument("--as-of", required=True, help="YYYY-MM-DD market date")
    p.add_argument("--reviewer", required=True, help="Username at capture/review")
    p.add_argument("--source", default="quantlib",
                   choices=["quantlib", "bloomberg-api", "bloomberg-screen", "intex", "formula", "textbook"])
    p.add_argument("--source-detail", default="", help="Free-form source description")
    p.add_argument("--out", required=True, type=Path, help="Output JSON path (absolute)")
    # Kind-specific args use --kw key=value to avoid argparse conflicts.
    p.add_argument("--kw", action="append", default=[],
                   help="Adapter-specific arg, repeat as needed: --kw tenor=5Y --kw rate=0.04")
    args_ns = p.parse_args()

    kind_args: dict[str, str] = {}
    for kv in args_ns.kw:
        if "=" not in kv:
            raise SystemExit(f"--kw expected key=value, got '{kv}'")
        k, v = kv.split("=", 1)
        kind_args[k] = v

    adapter = _load_adapter(args_ns.kind)
    args = AdapterArgs(
        as_of=args_ns.as_of,
        reviewer=args_ns.reviewer,
        out=args_ns.out.resolve(),
        source=args_ns.source,
        source_detail=args_ns.source_detail or f"adapter:{args_ns.kind}",
    )
    adapter.emit(args, kind_args)


if __name__ == "__main__":
    main()
```

```python
"""Adapter for --kind irs-par: vanilla par interest rate swap priced via QuantLib-Python.

CLI args (via --kw key=value):
  tenor=5Y           IRS tenor (years string, parsed by QL)
  rate=0.04          flat market rate (decimal, applied to both fixed + float)
  notional=10000000  notional, default 10M
  currency=USD       USD | EUR | GBP

The adapter builds a flat OIS curve at `rate`, prices a par swap (fixed=rate),
and emits npv, par_rate, dv01.
"""
from __future__ import annotations

from datetime import date
from typing import Any

try:
    import QuantLib as ql
except ImportError as e:
    raise SystemExit(
        "QuantLib-Python is required for --kind irs-par. "
        "Install with: uv pip install QuantLib"
    ) from e

from .base import BaseAdapter


_CCY_TO_INDEX = {
    "USD": ("USD-OIS", lambda y: ql.Sofr(y)),
    "EUR": ("EUR-ESTR", lambda y: ql.Estr(y)),
    "GBP": ("GBP-SONIA", lambda y: ql.Sonia(y)),
}


def _parse_tenor(s: str) -> ql.Period:
    """Convert '5Y' / '6M' to ql.Period."""
    return ql.Period(s)


class IrsParAdapter(BaseAdapter):
    @property
    def kind(self) -> str:
        return "irs-par"

    @property
    def domain(self) -> str:
        return "rates.irs"

    def build(self, args, kind_args: dict[str, Any]) -> dict[str, Any]:
        tenor = kind_args.get("tenor", "5Y")
        rate = float(kind_args.get("rate", "0.04"))
        notional = float(kind_args.get("notional", "10000000"))
        currency = kind_args.get("currency", "USD")
        if currency not in _CCY_TO_INDEX:
            raise ValueError(f"currency must be one of {sorted(_CCY_TO_INDEX)}, got {currency!r}")
        index_id, index_factory = _CCY_TO_INDEX[currency]

        # --- QuantLib setup ---------------------------------------------------
        as_of_d = date.fromisoformat(args.as_of)
        as_of_ql = ql.Date(as_of_d.day, as_of_d.month, as_of_d.year)
        ql.Settings.instance().evaluationDate = as_of_ql

        flat_curve = ql.FlatForward(as_of_ql, rate, ql.Actual365Fixed())
        flat_curve.enableExtrapolation()
        ts_handle = ql.YieldTermStructureHandle(flat_curve)
        index = index_factory(ts_handle)

        start = as_of_ql
        period = _parse_tenor(tenor)
        end = as_of_ql + period
        fixed_schedule = ql.MakeSchedule(start, end, ql.Period("6M"))
        float_schedule = ql.MakeSchedule(start, end, ql.Period("3M"))
        swap = ql.VanillaSwap(
            ql.VanillaSwap.Payer,
            notional,
            fixed_schedule,
            rate,
            ql.Thirty360(ql.Thirty360.BondBasis),
            float_schedule,
            index,
            0.0,
            ql.Actual360(),
        )
        engine = ql.DiscountingSwapEngine(ts_handle)
        swap.setPricingEngine(engine)

        npv = swap.NPV()
        par_rate = swap.fairRate()
        # DV01 via parallel +1bp bump (re-build curve at rate + 1bp).
        bumped = ql.FlatForward(as_of_ql, rate + 0.0001, ql.Actual365Fixed())
        bumped.enableExtrapolation()
        bumped_handle = ql.YieldTermStructureHandle(bumped)
        bumped_index = index_factory(bumped_handle)
        bumped_swap = ql.VanillaSwap(
            ql.VanillaSwap.Payer,
            notional,
            fixed_schedule,
            rate,
            ql.Thirty360(ql.Thirty360.BondBasis),
            float_schedule,
            bumped_index,
            0.0,
            ql.Actual360(),
        )
        bumped_swap.setPricingEngine(ql.DiscountingSwapEngine(bumped_handle))
        dv01 = bumped_swap.NPV() - npv

        # --- Build fixture body ----------------------------------------------
        # Inputs: a flat curve described as (year_fraction, discount_factor) knots
        # at standard pillars. The runner rebuilds finstack curves from these knots.
        years = [0.0, 0.25, 0.5, 1.0, 2.0, 3.0, 5.0, 7.0, 10.0]
        knots = [[t, float(flat_curve.discount(as_of_ql + ql.Period(int(t * 365), ql.Days)))]
                 for t in years]

        name = f"{currency.lower()}_irs_{('sofr' if currency=='USD' else 'estr' if currency=='EUR' else 'sonia')}_{tenor.lower()}_par"
        return {
            "name": name,
            "description": (
                f"{tenor} {currency} par swap, flat curve at {rate*100:.2f}%. "
                f"Fixed leg = {rate*100:.2f}%, semi-annual, 30/360. "
                f"Float leg = {currency} O/N compounded, quarterly, Act/360."
            ),
            "inputs": {
                "as_of_date": args.as_of,
                "curves": [
                    {"id": index_id, "kind": "discount", "knots": knots}
                ],
                "instrument": {
                    "kind": "irs",
                    "notional": notional,
                    "currency": currency,
                    "fixed_rate": rate,
                    "tenor": tenor,
                    "side": "pay_fixed",
                },
            },
            "expected_outputs": {
                "npv": float(npv),
                "par_rate": float(par_rate),
                "dv01": float(dv01),
            },
            "tolerances": {
                "npv": {"abs": 0.01, "rel": 1e-6},
                "par_rate": {"abs": 1e-7},
                "dv01": {"abs": 0.5, "rel": 1e-4},
            },
        }


ADAPTER = IrsParAdapter()
```

```rust
//! Domain runner: rates.irs — interest rate swap pricing goldens.

use crate::golden::runner::DomainRunner;
use crate::golden::schema::GoldenFixture;
use finstack_core::currency::Currency;
use finstack_core::dates::Date;
use finstack_core::market_data::context::MarketContext;
use finstack_core::market_data::term_structures::{DiscountCurve, ForwardCurve};
use finstack_core::math::interp::InterpStyle;
use finstack_core::money::Money;
use finstack_valuations::instruments::rates::irs::PayReceive;
use finstack_valuations::instruments::{Instrument, PricingOptions};
use finstack_valuations::metrics::MetricId;
use serde::Deserialize;
use std::collections::BTreeMap;
use std::str::FromStr;

#[derive(Debug, Deserialize)]
struct IrsInputs {
    as_of_date: String,
    curves: Vec<CurveSpec>,
    instrument: InstrumentSpec,
}

#[derive(Debug, Deserialize)]
struct CurveSpec {
    id: String,
    kind: String,
    knots: Vec<[f64; 2]>,
}

#[derive(Debug, Deserialize)]
struct InstrumentSpec {
    kind: String,
    notional: f64,
    currency: String,
    fixed_rate: f64,
    tenor: String,
    side: String,
}

fn parse_currency(s: &str) -> Result<Currency, String> {
    Currency::from_str(s).map_err(|e| format!("invalid currency '{}': {}", s, e))
}

fn parse_side(s: &str) -> Result<PayReceive, String> {
    match s {
        "pay_fixed" => Ok(PayReceive::PayFixed),
        "receive_fixed" => Ok(PayReceive::ReceiveFixed),
        other => Err(format!("invalid side '{}'; expected pay_fixed | receive_fixed", other)),
    }
}

fn parse_iso_date(s: &str) -> Result<Date, String> {
    // YYYY-MM-DD
    let parts: Vec<&str> = s.split('-').collect();
    if parts.len() != 3 {
        return Err(format!("invalid date '{}'; expected YYYY-MM-DD", s));
    }
    let y: i32 = parts[0].parse().map_err(|e| format!("invalid year: {}", e))?;
    let m: u8 = parts[1].parse().map_err(|e| format!("invalid month: {}", e))?;
    let d: u8 = parts[2].parse().map_err(|e| format!("invalid day: {}", e))?;
    let month = time::Month::try_from(m).map_err(|e| format!("invalid month: {}", e))?;
    Date::from_calendar_date(y, month, d).map_err(|e| format!("invalid date: {}", e))
}

fn parse_tenor_years(s: &str) -> Result<i32, String> {
    let trimmed = s.trim_end_matches('Y').trim_end_matches('y');
    trimmed.parse().map_err(|e| format!("invalid tenor '{}': {}", s, e))
}

fn build_discount_curve(spec: &CurveSpec, as_of: Date) -> Result<DiscountCurve, String> {
    let knots: Vec<(f64, f64)> = spec.knots.iter().map(|k| (k[0], k[1])).collect();
    DiscountCurve::builder(&spec.id)
        .base_date(as_of)
        .knots(knots)
        .interp(InterpStyle::Linear)
        .build()
        .map_err(|e| format!("build discount curve {}: {}", spec.id, e))
}

fn build_forward_curve(spec: &CurveSpec, as_of: Date) -> Result<ForwardCurve, String> {
    let knots: Vec<(f64, f64)> = spec.knots.iter().map(|k| (k[0], k[1])).collect();
    ForwardCurve::builder(&spec.id, 0.25)
        .base_date(as_of)
        .knots(knots)
        .interp(InterpStyle::Linear)
        .build()
        .map_err(|e| format!("build forward curve {}: {}", spec.id, e))
}

pub struct IrsRunner;

impl DomainRunner for IrsRunner {
    fn run(&self, fixture: &GoldenFixture) -> Result<BTreeMap<String, f64>, String> {
        let inputs: IrsInputs = serde_json::from_value(fixture.inputs.clone())
            .map_err(|e| format!("parse inputs: {}", e))?;
        if inputs.instrument.kind != "irs" {
            return Err(format!(
                "instrument.kind must be 'irs', got '{}'",
                inputs.instrument.kind
            ));
        }

        let as_of = parse_iso_date(&inputs.as_of_date)?;
        let currency = parse_currency(&inputs.instrument.currency)?;
        let side = parse_side(&inputs.instrument.side)?;
        let years = parse_tenor_years(&inputs.instrument.tenor)?;
        let end = Date::from_calendar_date(as_of.year() + years, as_of.month(), as_of.day())
            .map_err(|e| format!("compute end date: {}", e))?;

        // Market context: insert each curve. Default convention requires both a
        // discount and a forward curve for vanilla IRS pricing in finstack.
        // The fixture must include both; if the adapter only emitted discount,
        // mirror it as the forward curve at the same flat rate.
        let mut market = MarketContext::new();
        let mut have_discount = false;
        let mut have_forward = false;
        for spec in &inputs.curves {
            match spec.kind.as_str() {
                "discount" => {
                    market = market.insert(build_discount_curve(spec, as_of)?);
                    have_discount = true;
                    // Auto-mirror to a forward curve named USD-SOFR-3M etc. for
                    // single-curve test fixtures. The adapter today only emits
                    // discount knots; add a forward curve at flat rate inferred
                    // from the first knot.
                    if !have_forward {
                        // Convert discount to flat rate: pull `r` from the second
                        // knot via -ln(df) / t when knots come from a flat curve.
                        let (t1, df1) = (spec.knots[1][0], spec.knots[1][1]);
                        let r = if t1 > 0.0 { -df1.ln() / t1 } else { 0.0 };
                        let fwd_id = match currency {
                            Currency::USD => "USD-SOFR-3M",
                            Currency::EUR => "EUR-ESTR-3M",
                            Currency::GBP => "GBP-SONIA-3M",
                            _ => return Err(format!("unsupported currency {:?}", currency)),
                        };
                        let fwd_spec = CurveSpec {
                            id: fwd_id.to_string(),
                            kind: "forward".to_string(),
                            knots: vec![[0.0, r], [10.0, r]],
                        };
                        market = market.insert(build_forward_curve(&fwd_spec, as_of)?);
                        have_forward = true;
                    }
                }
                "forward" => {
                    market = market.insert(build_forward_curve(spec, as_of)?);
                    have_forward = true;
                }
                other => return Err(format!("curve.kind must be 'discount' or 'forward', got '{}'", other)),
            }
        }
        if !have_discount {
            return Err("fixture must include at least one discount curve".to_string());
        }

        let swap = crate::finstack_test_utils::usd_irs_swap(
            format!("IRS-GOLDEN-{}", fixture.name),
            Money::new(inputs.instrument.notional, currency),
            inputs.instrument.fixed_rate,
            as_of,
            end,
            side,
        )
        .map_err(|e| format!("build swap: {}", e))?;

        let metrics_needed: Vec<MetricId> = fixture
            .expected_outputs
            .keys()
            .filter_map(|k| match k.as_str() {
                "npv" => None, // value() returns NPV directly
                "par_rate" => Some(MetricId::ParRate),
                "dv01" => Some(MetricId::Dv01),
                _ => None,
            })
            .collect();

        let result = swap
            .price_with_metrics(&market, as_of, &metrics_needed, PricingOptions::default())
            .map_err(|e| format!("price swap: {}", e))?;

        let mut out = BTreeMap::new();
        for metric in fixture.expected_outputs.keys() {
            let value = match metric.as_str() {
                "npv" => result.value.amount(),
                "par_rate" => *result
                    .measures
                    .get(MetricId::ParRate.as_str())
                    .ok_or("ParRate metric missing")?,
                "dv01" => *result
                    .measures
                    .get(MetricId::Dv01.as_str())
                    .ok_or("Dv01 metric missing")?,
                other => return Err(format!("unknown metric '{}' for rates.irs domain", other)),
            };
            out.insert(metric.clone(), value);
        }
        Ok(out)
    }
}
```

```python
"""Domain runner: rates.irs — IRS pricing goldens via finstack PyO3 bindings."""
from __future__ import annotations

from datetime import date
from typing import Any

import finstack
from finstack import valuations as fsv

from ..schema import GoldenFixture


_CCY_FORWARD_ID = {
    "USD": "USD-SOFR-3M",
    "EUR": "EUR-ESTR-3M",
    "GBP": "GBP-SONIA-3M",
}


def _parse_iso_date(s: str) -> date:
    return date.fromisoformat(s)


def _parse_tenor_years(s: str) -> int:
    return int(s.rstrip("Yy"))


def run(fixture: GoldenFixture) -> dict[str, float]:
    inputs = fixture.inputs
    inst = inputs["instrument"]
    if inst["kind"] != "irs":
        raise ValueError(f"instrument.kind must be 'irs', got {inst['kind']!r}")

    as_of = _parse_iso_date(inputs["as_of_date"])
    currency = inst["currency"]
    notional = float(inst["notional"])
    fixed_rate = float(inst["fixed_rate"])
    tenor_years = _parse_tenor_years(inst["tenor"])
    end = date(as_of.year + tenor_years, as_of.month, as_of.day)
    side = inst["side"]

    market = fsv.MarketContext()
    have_forward = False
    for c in inputs["curves"]:
        if c["kind"] == "discount":
            curve = fsv.DiscountCurve.builder(c["id"]) \
                .base_date(as_of) \
                .knots([(t, df) for t, df in c["knots"]]) \
                .interp("Linear") \
                .build()
            market = market.insert(curve)
            if not have_forward:
                # Mirror to a forward curve at flat rate inferred from the
                # second knot (matches the Rust runner's auto-mirror).
                t1, df1 = c["knots"][1]
                r = -((df1 ** 0) and __import__("math").log(df1)) / t1 if t1 > 0 else 0.0
                fwd_id = _CCY_FORWARD_ID[currency]
                fwd = fsv.ForwardCurve.builder(fwd_id, 0.25) \
                    .base_date(as_of) \
                    .knots([(0.0, r), (10.0, r)]) \
                    .interp("Linear") \
                    .build()
                market = market.insert(fwd)
                have_forward = True
        elif c["kind"] == "forward":
            curve = fsv.ForwardCurve.builder(c["id"], 0.25) \
                .base_date(as_of) \
                .knots([(t, r) for t, r in c["knots"]]) \
                .interp("Linear") \
                .build()
            market = market.insert(curve)
            have_forward = True
        else:
            raise ValueError(f"curve.kind must be 'discount' or 'forward', got {c['kind']!r}")

    swap = fsv.usd_irs_swap(
        id=f"IRS-GOLDEN-{fixture.name}",
        notional=fsv.Money(notional, currency),
        fixed_rate=fixed_rate,
        start=as_of,
        end=end,
        side=("PayFixed" if side == "pay_fixed" else "ReceiveFixed"),
    )

    metrics_needed = []
    if "par_rate" in fixture.expected_outputs:
        metrics_needed.append("ParRate")
    if "dv01" in fixture.expected_outputs:
        metrics_needed.append("Dv01")

    result = swap.price_with_metrics(market, as_of, metrics_needed)

    out: dict[str, float] = {}
    for metric in fixture.expected_outputs:
        if metric == "npv":
            out[metric] = float(result.value.amount())
        elif metric == "par_rate":
            out[metric] = float(result.measures["par_rate"])
        elif metric == "dv01":
            out[metric] = float(result.measures["dv01"])
        else:
            raise ValueError(f"unknown metric {metric!r} for rates.irs domain")
    return out
```
